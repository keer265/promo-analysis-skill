# CHECKLIST

本文件用于在交付前自检，确保分析与报告成体系、口径一致、结论可落地。

---

## A-1. 调用配置检查

- [ ] 是否先确认了数据路径/数据来源
- [ ] 若用户未给路径/来源，是否先询问
- [ ] 是否确认了输出格式（Word / HTML / 两者都要）
- [ ] 是否确认了输出路径
- [ ] 是否确认了必要选择项（如 HTML 风格）
- [ ] 用户已给出的信息是否被复用而非重复追问

---

## A0. 阶段闸门检查

- [ ] 是否在正式分析前做了可执行性判断
- [ ] 是否列出当前已读取到的表与关键字段
- [ ] 是否判断了当前执行等级（Level 0~4）
- [ ] 是否列出“能做 / 不能做 / 降级做”
- [ ] 是否在闸门后明确询问用户：继续 / 仅输出缺口 / 停止
- [ ] 若用户未选择继续，是否避免直接进入完整分析

---

## A. 输入检查

- [ ] 是否明确有哪些表
- [ ] 是否确认字段名与含义
- [ ] 是否说明数据覆盖时间范围
- [ ] 是否检查日期字段可解析
- [ ] 是否检查主键可关联
- [ ] 是否检查负金额、负优惠、重复记录
- [ ] 是否检查活动表按日期可唯一化

---

## B. 口径检查

- [ ] GMV 口径是否统一
- [ ] 优惠金额口径是否统一
- [ ] Pay UV / Purchase UV / Exposure UV 是否混用
- [ ] 活动日定义是否清晰
- [ ] 活动窗口定义是否清晰
- [ ] 基线样本是否剔除 buffer

---

## C. 方法链完整性检查

- [ ] 是否做了描述性统计
- [ ] 是否做了周期性识别
- [ ] 是否构建了反事实基线
- [ ] 是否输出了活动窗口增量
- [ ] 是否解释了事件窗口动态
- [ ] 是否做了至少一种稳健性 / 准实验验证
- [ ] 是否识别了提前消费或蚕食
- [ ] 是否做了用户分层或商品敏感度分析
- [ ] 是否给出了预算优化方案

---

## D. 图表检查

- [ ] 每张图是否有标题
- [ ] 每张图是否说明口径
- [ ] 每张图后是否有读图指引
- [ ] 每张图后是否有关键洞察
- [ ] 是否避免只放图不解释
- [ ] 是否避免图与正文结论相矛盾

---

## E. 关键数值检查

- [ ] 实际 GMV、基线 GMV、增量 GMV 是否逻辑一致
- [ ] 增量率公式是否一致
- [ ] iROI 公式是否一致
- [ ] 百分比是否与分子分母匹配
- [ ] 汇总值与分项值是否大致一致
- [ ] 表格与正文中的数字是否一致

---

## F. 结论检查

- [ ] 是否区分“整体均值正向”与“局部异质性”
- [ ] 是否说明活动是否主要来自流量扩张
- [ ] 是否说明效率是否改善
- [ ] 是否说明是否存在回吐
- [ ] 是否说明折扣深度不是唯一决定因素
- [ ] 是否避免过强因果表述

---

## G. 商业建议检查

- [ ] 每条建议是否能回溯到分析证据
- [ ] 是否区分用户、商品、类目、时间窗口
- [ ] 是否避免“一刀切加深折扣”
- [ ] 是否给出优先级
- [ ] 是否说明预期效果是“方向性”还是“已验证”
- [ ] 是否给出 A/B 验证思路

---

## H. 输出形式检查

### Word
- [ ] 目录是否完整
- [ ] 章节结构是否合理
- [ ] 表格与标题层级是否统一
- [ ] 是否适合继续修改与批注

### HTML
- [ ] 是否只保留核心结果、核心数据、核心发现、核心图以及建议
- [ ] 是否为可编辑 HTML 文件
- [ ] 是否适合浏览器保存为 PDF
- [ ] 页面说明框是否齐全

---

## I. 可复用性检查

- [ ] 是否写清了输入要求
- [ ] 是否写清了字段依赖
- [ ] 是否写清了方法适用条件
- [ ] 是否写清了数据不足时的降级方案
- [ ] 是否写清了后续可扩展位置

---

## J. 对话框检查输出

- [ ] 是否补充了 AI 幻觉/事实一致性检查
- [ ] 是否指出了证据不足项
- [ ] 是否指出了需要人工复核的数字或图表
- [ ] 是否确认无占位符残留
- [ ] 这些检查是否只输出在对话框而非写入文件

---

## K. 最终交付前最后一问

在交付前，请自问：

1. 如果换一批活动数据，这套结构还能不能直接复用？
2. 如果没有曝光表，这套报告会不会误导？
3. 如果只有整体均值为正，但近一半 SKU 无增长，报告有没有把这种矛盾讲清楚？
4. Word 和 HTML 的内容层级是否已经明显区分？
5. HTML 是否真的能被继续修改并导出为 PDF？
